# Emacs

| 操作    | キー                                           | コマンド名 |
| :------ | :--------------------------------------------- | :--------- |
| C-      | Ctrl                                           |
| M-      | Alt                                            |
| C-h t   | チュートリアル                                 |
| C-h C-h | ヘルプタイプ一覧                               |
| C-x o   | ウインドウ移動                                 |
| C-x 0   | カレントウインドウを閉じる(バッファは閉じない) |
