# Python

## 関数の引数リストにあるスラッシュ (/) は何を意味しますか？
引数リストの末尾にスラッシュは、この2つの仮引数が両方とも位置専用引数であることを意味します
https://docs.python.org/ja/3/faq/programming.html#what-does-the-slash-in-the-parameter-list-of-a-function-mean
