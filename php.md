# PHP

## 関数
- String 関数
  - nl2br ( string $string [, bool $is_xhtml = TRUE ] ) : string
    - 改行文字の前に HTML の改行タグを挿入する
  - strcmp ( string $str1 , string $str2 ) : int
    - バイナリセーフな文字列比較
  - ltrim ( string $str [, string $character_mask ] ) : string
    - 文字列の最初から空白 (もしくはその他の文字) を取り除く
