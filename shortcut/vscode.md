# vscode

| 内容                                                   | Windows            | Mac  |
| :----------------------------------------------------- | :----------------- | :--- |
| 新規ウインドウを開く                                   | Ctrl + Shift + n   |
| ファイル内全部折り畳み                                 | Ctrl + k Ctrl + 0  |
| ファイル内全部展開                                     | Ctrl + k Ctrl + j  |
| カーソル位置にアンカーをセット                         | Ctrl + k Ctrl + b  |
| アンカーからカーソル位置間を選択                       | Ctrl + k Ctrl + k  |
| アンカー解除                                           | Esc                |
| スマート選択                                           | Shift + Alt + ←, → |
| 選択されている・カーソル周辺と一致する文字列すべて選択 | Ctrl + Shift + l   |
| 置換                                                   | Ctrl + h           |
